
Transmission
############

Introduction
============

This is the transmission module. This module helps using python to connect to a Transmission_ json-RPC service. transmission module is compatible with Transmission 1.3 and later.

.. _Transmission: http://www.transmissionbt.com/

vxl is short for v√§xel, the swedish word for gearbox.

.. contents::
   :depth: 3

Getting started
===============

First download the package and extract the contents. Then install using setup.py.

::

    $ python setup.py install

This will install the transmission module and vxl cli.

Dependecies
-----------

 * simplejson 1.7.1 or later
